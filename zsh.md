# zsh

## git

| 别名 | 完整命令 | 说明 |
| :--- | :--- | :--- |
| `gs` | `git status` | 查看仓库状态（使用频率最高） |
| `ga` | `git add` | 添加文件到暂存区 |
| `gaa` | `git add --all` | 添加所有变更（包括删除的文件） |
| `gcam` | `git commit -a -m` | 超级常用：自动 add 并提交，后接消息 |
| `gl` | `git pull` | 拉取远程代码 |
| `gp` | `git push` | 推送代码到远程 |
| `glog` | `git log --oneline --graph --decorate` | 以图形化单行模式查看提交历史 |
| `gco` | `git checkout` | 切换分支或恢复文件 |
| `gb` | `git branch` | 列出本地分支 |
| `gm` | `git merge` | 合并分支 |
| `gba` | `git branch -a` | 列出所有分支（含远程） |
| `gbd` | `git branch -d` | 删除已合并的分支 |
| `gcob` | `git checkout -b` | 创建并切换到新分支 |
